# smart-cash-app

Merupakan Aplikasi pencatatan uang kas. baik digunakan untuk kalangan akademisi maupun organisasi dalam memanajemen uang kas.

Fitur-fitur :
1.  Input Kas
2.  Statistik Kas
3.  Export ke PDF
4.  Input Agenda

Supported :
1.  Bahasa Pemrograman : PHP, HTML, CSS, JS  
2.  Framework PHP : CodeIgniter              
3.  Bootstrap
4.  MySQL Databases
